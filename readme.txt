About:

This program was designed by Damian Mietus for the /r/dailyprogrammer challenge
#320 [Easy] Spiral Ascension.

Running the Program:

    - In the terminal traverse to the 'spiral' folder
    - type 'make' to compile the Programmer
    - the executable is in the 'bin' directory,
      so you must type './bin/spiral x' (where x is the number to be spiraled)
